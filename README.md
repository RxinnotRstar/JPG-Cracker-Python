# JPG 比特流破坏器

一个用于对 JPG 图片进行比特级损坏实验的 GUI 工具，支持区域选择、强度调节和批量保存。

---

## 功能特点

- **自动解析结构**：识别 JPG 文件格式并定位可破坏区域（跳过文件头 SOI/EOI）
- **区域选择**：支持 DQT/DHT/SOS/EXIF/COM 等段，默认选中最大区域
- **破坏强度**：可翻转 1–10 位比特
- **两种模式**：
  - 随机散选：在区域内随机翻转比特
  - 连续片段：翻转连续的比特段
- **内存操作**：全程在内存处理，不落盘
- **保存结果**：（可选）保存破坏后的图片和原图备份

---

## 使用说明

1. 运行脚本：
   ```bash
   python 破坏JPG.py
   ```
2. 点击「打开图片」加载 JPG 文件
3. 选择破坏区域、比特数和模式
4. 点击「执行破坏」查看效果
5. 勾选「保存图片」可将结果保存至 `./文件名前8位/` 目录

---

## 注意事项

- 仅支持 JPG 格式
- 破坏后图片可能无法解码，保存时可按需开启「过滤不可解码」选项
